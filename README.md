# Cape-Town-Municipal_Arrears_Suburbs_and_Service_Type_2025_to_2026


Municipal Arrears: Suburbs & Service Type Analysis (2025–2026)
## Project Overview

This project analyzes municipal billing and arrears data across different suburbs and service types for the 2025–2026 period. The objective is to evaluate revenue performance, identify high risk areas with outstanding debt, and provide data-driven insights to support financial decision-making and revenue recovery strategies.

## Key Objectives
Assess total arrears and billed revenue by suburb and service type
Calculate and analyze Arrears Percentage as a critical performance indicator
Identify trends, outliers, and underperforming areas
Compare billing efficiency across locations and time periods

## Tools & Technologies
SQL (Databricks): Data cleaning, transformation, and aggregation
Microsoft Excel: Initial data preparation and validation
Power BI / Tableau: Dashboard development and visualization
Miro: Project planning and workflow desig

## Key Metrics (KPIs)
Total Arrears
Total Billed Amount
Average Arrears
Arrears Percentage (%)
Number of Records Accounts

## Dashboard Features
Arrears by suburb (bar chart)
Revenue trends over time (line chart)
KPI summary cards for quick insights
Filters for service type, location, and time period
## Insights & Impact

The analysis highlights high-arrears suburbs, inefficiencies in billing, and areas requiring targeted intervention. These insights can help municipalities improve revenue collection, optimize billing processes, and reduce financial risk.
